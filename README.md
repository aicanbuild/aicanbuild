我是 **Can**，一个独立开发者。

`aicanbuild` 是我的个人厂牌——一个不断打造 AI 小工具的"作坊"。每件作品都只解决一个具体的问题，开源、无追踪、用心做。

**当前产品**：[TubeMask · 油管装忙](https://github.com/aicanbuild/tubemask)——把 YouTube 伪装成工作页面的 Chrome 插件，让屏幕一眼看上去像在看文档、仪表盘、文章或邮件。

**做事原则**：

- 一次一个产品，做完再做下一个
- 中英文 README 同步
- 一律 MIT 协议
- 不上任何追踪 / 统计

**联系方式**：Twitter [@aicanbuild](https://x.com/aicanbuild)，或者在任意项目的 issues 里打招呼。
